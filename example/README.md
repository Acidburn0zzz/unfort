### Unfort example


## Installation

```
npm install --save \
  unfort \
  babel-plugin-hot-swap-declarative-modules \
  babel-plugin-transform-es2015-modules-commonjs \
  express \
  lodash \
  socket.io \
  strip-ansi
```


### Run

```
node unfort.js
```

Visit: http://127.0.0.1:3000/

Check the console, and make some changes to `src/main.js`.
